**Time, Speed & Distance** — MAT Study Guide

**Fundamental Formula:**
```
Speed = Distance / Time
Distance = Speed × Time
Time = Distance / Speed
```

**Unit Consistency — CRITICAL:**
- If speed is in km/hr and time in minutes → convert time to hours first
- **1 km/hr = (5/18) m/sec**
- **1 m/sec = (18/5) km/hr = 3.6 km/hr**

**Shortcut for Conversion:**
- km/hr to m/sec: multiply by 5/18
- m/sec to km/hr: multiply by 18/5
- Example: 72 km/hr = 72 × (5/18) = 20 m/sec ✓

**Average Speed (Weighted Average Concept):**
For a journey with equal distances at different speeds:
- **Average Speed = Total Distance / Total Time**
- If distance d is covered at speed v₁ and same distance d at speed v₂:
  **Average Speed = (2 × v₁ × v₂) / (v₁ + v₂)**

**Shortcut:** Harmonic mean of the two speeds

**For equal time periods:**
- Average speed = (v₁ + v₂) / 2 (simple arithmetic mean)

**Relative Speed — THE KEY CONCEPT:**

| Scenario | Relative Speed |
|----------|---------------|
| Moving towards each other | v₁ + v₂ |
| Moving away from each other | v₁ + v₂ |
| Moving in same direction (faster catches slower) | v₁ - v₂ |
| Same direction (approaching from behind) | v₁ - v₂ |

**Meeting Point Problems:**
If two persons A and B start from points P and Q towards each other:
- They meet when: Distance covered by A + Distance covered by B = Total distance
- Time to meet = Total Distance / (v_A + v_B)

**Train Problems:**
- **Train passing a stationary person:** Time = Length of train / Speed of train
- **Train passing another train:** Time = (L₁ + L₂) / (v₁ + v₂) [if opposite directions]
- **Train passing a platform:** Time = (Length of train + Length of platform) / Speed
- **Two trains crossing each other going same direction:** Time = (L₁ + L₂) / |v₁ - v₂|

**Boats and Streams:**
- Downstream speed = u + v (boat speed + stream speed)
- Upstream speed = u - v (boat speed - stream speed)
- **Speed of boat in still water = (Downstream + Upstream) / 2**
- **Speed of stream = (Downstream - Upstream) / 2**

**Circular Motion:**
- When two runners start from same point on a circular track:
- Time to meet at starting point = LCM(time taken by each to complete one round)
- If running in opposite directions: relative speed = v₁ + v₂

**Common Mistakes to Avoid:**
1. Forgetting to convert units (km/hr vs m/sec)
2. Adding speeds instead of subtracting in same-direction problems
3. Confusing average speed formula for equal distance vs equal time
4. Mixing up upstream/downstream in boats and streams

**MAT Shortcut — The "Catch-Up" Method:**
When body A (faster) chases body B (slower) starting t hours later:
- Distance covered by A in time T = Distance covered by B in time (T + t)
- Solve: v_A × T = v_B × (T + t)

**Example Question:**
Two trains 180m and 220m long run at speeds of 46 km/hr and 62 km/hr respectively, in opposite directions. Find the time taken to cross each other.
Solution:
- Total length = 180 + 220 = 400m
- Relative speed = 46 + 62 = 108 km/hr
- Convert: 108 × (5/18) = 30 m/sec
- Time = 400/30 = **13.33 seconds** ✓

**Time, Speed & Distance** — Comprehensive MAT Notes

**Section 1: Foundational Concepts**

**The Basic Relationship:**
Speed is a rate — it tells us how fast something is moving. The fundamental equation:
**Distance = Rate × Time**

This triangle method helps remember:
```
    Distance
   ───────── = Speed
     Time
```

**Understanding Units:**
- Speed is measured as distance per unit time
- Common units: m/sec, km/hr, miles/hr, ft/sec
- Always ensure units are consistent before calculation

**Conversion Factors (Memorize for MAT):**
| From | To | Multiply by |
|------|-----|-------------|
| km/hr | m/sec | 5/18 |
| m/sec | km/hr | 18/5 |
| km/hr | miles/hr | 0.621 |
| miles/hr | km/hr | 1.609 |

**Why 5/18 works:**
1 km = 1000 m, 1 hr = 3600 sec
Therefore: 1 km/hr = 1000/3600 m/sec = 5/18 m/sec

**Section 2: Average Speed — Deep Dive**

**Case 1: Equal Distances at Different Speeds**
A person travels from A to B at 40 km/hr and returns at 60 km/hr. Find average speed.

**Method 1 (Formula):**
Average Speed = (2 × v₁ × v₂) / (v₁ + v₂)
= (2 × 40 × 60) / (40 + 60)
= 4800 / 100 = **48 km/hr**

**Method 2 (Fundamental):**
Let distance AB = d
- Time going = d/40
- Time returning = d/60
- Total distance = 2d
- Total time = d/40 + d/60 = (3d + 2d)/120 = 5d/120 = d/24
- Average speed = 2d / (d/24) = **48 km/hr** ✓

**Key Insight:** Average speed is NOT the arithmetic mean when distances are equal — it's the harmonic mean.

**Case 2: Equal Time Periods**
If a car travels at 60 km/hr for 2 hours and at 80 km/hr for 2 hours:
- Total distance = 60×2 + 80×2 = 120 + 160 = 280 km
- Total time = 4 hours
- Average speed = 280/4 = **70 km/hr** = (60 + 80)/2 ✓

**Case 3: Multiple Legs with Different Distances**
Always use: **Average Speed = Total Distance / Total Time**

**Section 3: Relative Speed Mastery**

**Concept 1: Bodies Moving Towards Each Other**
Two cars start from points A and B (200 km apart), moving towards each other at 50 km/hr and 70 km/hr. When and where do they meet?

**Method:**
Relative speed = 50 + 70 = 120 km/hr
Time to meet = Distance / Relative speed = 200/120 = 5/3 hours = 1 hr 40 min
Distance from A = 50 × (5/3) = 250/3 = **83.33 km** ✓

**Concept 2: Bodies Moving in Same Direction**
A man starts from point A at 8 km/hr. After 2 hours, another man starts from A at 12 km/hr in the same direction. When does the second man catch up?

**Catch-Up Formula:**
Distance covered by first man in 2 hours = 8 × 2 = 16 km
Relative speed = 12 - 8 = 4 km/hr
Time to catch = 16/4 = **4 hours**
Catch-up point = 12 × 4 = **48 km from A** ✓

**Concept 3: One Starts Later from Different Point**
Point A is 60 km ahead. Person X starts from A at 40 km/hr. Person Y starts from starting point 2 hours later at 60 km/hr. When does Y catch X?

- In 2 hours, X covers = 40 × 2 = 80 km
- Now X is at A + 80 km, Y at starting point
- Gap = 80 km
- Relative speed = 60 - 40 = 20 km/hr
- Time to catch = 80/20 = 4 hours
- Y catches X after 4 more hours = 6 hours from start ✓

**Section 4: Train Problems — Comprehensive**

**Principle:** When a train passes an object, the distance covered equals the train's length.

**Type 1: Train Passing a Stationary Person**
A 150m train passes a man standing on a platform at 45 km/hr. Time taken?
- Speed = 45 × (5/18) = 12.5 m/sec
- Time = 150/12.5 = **12 seconds** ✓

**Type 2: Train Passing Another Train (Opposite Directions)**
Two trains 200m and 150m long cross each other while running at 60 km/hr and 90 km/hr in opposite directions. Time to cross?

- Combined length = 350m
- Relative speed = 60 + 90 = 150 km/hr = 150 × (5/18) = 41.67 m/sec
- Time = 350/41.67 = **8.4 seconds** ✓

**Type 3: Train Passing Another Train (Same Direction)**
Two trains 180m and 220m long run at 50 km/hr and 70 km/hr in same direction. How long for faster train to completely pass the slower?

- Relative speed = 70 - 50 = 20 km/hr = 20 × (5/18) = 5.56 m/sec
- Total distance = 180 + 220 = 400m
- Time = 400/5.56 = **72 seconds** ✓

**Type 4: Train Passing a Platform**
A 200m train crosses a 300m platform at 72 km/hr. Time?
- Distance = 200 + 300 = 500m
- Speed = 72 × (5/18) = 20 m/sec
- Time = 500/20 = **25 seconds** ✓

**Section 5: Boats and Streams**

**Stream Effect:**
- **Downstream:** Boat moves WITH the stream → speeds ADD
- **Upstream:** Boat moves AGAINST the stream → speeds SUBTRACT

**Standard Notation:**
- u = Speed of boat in still water (in km/hr or m/sec)
- v = Speed of stream
- Downstream speed (D) = u + v
- Upstream speed (U) = u - v

**Key Formulas:**
- u = (D + U) / 2
- v = (D - U) / 2

**Example:**
A boat goes downstream in 4 hours and upstream in 6 hours, covering 48 km each way. Find speeds.
- Downstream speed = 48/4 = 12 km/hr
- Upstream speed = 48/6 = 8 km/hr
- Boat in still water = (12 + 8)/2 = **10 km/hr** ✓
- Stream speed = (12 - 8)/2 = **2 km/hr** ✓

**Section 6: Problem-Solving Frameworks for MAT**

**Framework 1: The "Gap" Method**
When two moving objects create or close a gap:
1. Find initial gap
2. Find rate of gap change (relative speed)
3. Gap closed when relative distance covered = initial gap

**Framework 2: The "Meeting Point" Method**
For two persons starting from two points:
1. Let meeting time = t
2. Distance₁ + Distance₂ = Total Distance
3. v₁t + v₂t = D → t = D/(v₁ + v₂)

**Framework 3: The "Round Trip" Method**
For journeys with return:
1. Calculate time for each leg
2. Total time = sum of all legs
3. Average speed = (2 × d) / total time

**Section 7: MAT-Specific Strategies**

**Mental Math Shortcuts:**
- To find time: Distance/Speed — convert speed to easy numbers first
- If speed is awkward, convert distance to match
- Example: 36 km/hr = 10 m/sec (nice round number!)

**Elimination Techniques:**
1. If answer options have very different speeds, estimate first
2. Check if your answer makes sense dimensionally
3. Use the fact that time = Distance/Speed inversely

**Common Question Patterns in MAT:**
1. "A man covers X km in Y hours. Find speed" — Direct application
2. "Two trains A and B start..." — Relative speed problems
3. "A boat goes downstream/upstream..." — Stream problems
4. "Average speed for whole journey..." — Weighted average
5. "A train passes a pole/platform..." — Length + speed

**Tricks and Shortcuts:**
1. **For km/hr to m/sec:** Just multiply by 5 and divide by 18
2. **For equal distance average speed:** Use harmonic mean formula
3. **For trains crossing:** Add lengths, use relative speed
4. **For boats:** Remember "with current = +, against = -"
