## Coding-Decoding — Quick Reference

### Key Pattern/Rule
Find the pattern that maps the given word to its code, then apply the same pattern to decode or encode the new word.

### Memory Trick
**"Spot the shift, count the gap, apply it right."** Always check your pattern against ALL given examples before answering.

### 1-Sentence Summary
Coding-decoding asks you to find an underlying rule (letter shift, number pattern, or word transformation) and apply it to a new input.

### Quick Example
Q: In a certain code, COMPUTER is written as RFUVQNPC. How is MEDICINE written?
A: **MNFJDJOF** — Each letter is shifted -1 (C→B, O→N, etc.). Applying to MEDICINE gives MNFJDJOF.
