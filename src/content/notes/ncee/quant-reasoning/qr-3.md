# Number Analogies

### ЁЯЯв Lite тАФ Quick Review (1hтАУ1d)
> Rapid summary for last-minute revision before your exam.

Number Analogies appear in the NCEE Quantitative Reasoning section and test your ability to recognise numerical relationships. A typical question looks like this:

**"12 : 144 :: 5 : ?**

You must find the relationship between 12 and 144, then apply the same relationship to 5 to find the answer.

**Common Relationship Types:**

| Relationship | Example | Pattern |
|---|---|---|
| Square | 6 : 36 | $6^2 = 36$ |
| Cube | 3 : 27 | $3^3 = 27$ |
| Multiply by itself | 9 : 81 | $9 \times 9 = 81$ |
| Double | 7 : 14 | $7 \times 2 = 14$ |
| Add a constant | 3 : 7 | $+4$ each time |

**тЪб NCEE Exam Tip:** Always check if the first pair involves multiplication, division, addition, subtraction, squaring, or cubing before applying to the second number.

**Detailed Breakdown of Number Analogy Types**

Number analogies require you to identify the mathematical operation connecting the first pair of numbers, then apply that same operation to find the missing number.

**Type 1: Square/Cube Relationships**

- $4 : 16$ тЖТ $4^2 = 16$
- $3 : 27$ тЖТ $3^3 = 27$
- $7 : 49$ тЖТ $7^2 = 49$
- $5 : 125$ тЖТ $5^3 = 125$

**Type 2: Arithmetic Operations**

*Addition:* $8 : 11$ тЖТ $+3$ тЖТ so $15 : 18$
*Subtraction:* $20 : 15$ тЖТ $-5$ тЖТ so $12 : 7$
*Multiplication:* $6 : 30$ тЖТ $\times 5$ тЖТ so $8 : 40$
*Division:* $45 : 9$ тЖТ $\div 5$ тЖТ so $35 : 7$

**Type 3: Ratio and Proportion**

$3 : 9 :: 4 : 12$ тЖТ Ratio is $1:3$
$5 : 20 :: 7 : 28$ тЖТ Multiply by 4

**Type 4: Mixed Operations**

Sometimes the relationship involves multiple steps:
- $6 : 35$ тЖТ $6 \times 6 - 1 = 35$
- $8 : 63$ тЖТ $8 \times 8 - 1 = 63$

**тЪб NCEE Strategy:**
1. Look at the first number in the first pair тАФ what can you do to it to get the second?
2. Test your hypothesis on the second pair
3. If it works, apply to the question number
4. If it doesn't, try another operation

**Advanced Number Analogy Patterns and Problem-Solving**

Mastering number analogies requires systematic practice and recognition of increasingly complex patterns.

**Pattern Recognition Hierarchy:**

When approaching a number analogy problem, test relationships in this order:

**Level 1 тАФ Direct Operations**
1. Is B = A + constant?
2. Is B = A - constant?
3. Is B = A ├Ч constant?
4. Is B = A ├╖ constant?

**Level 2 тАФ Power Relationships**
5. Is B = $A^2$?
6. Is B = $A^3$?
7. Is B = $\sqrt{A}$?

**Level 3 тАФ Composite Operations**
8. Is B = $(A \times \text{constant}) \pm \text{constant}$?
9. Is B = $(A^2) \pm A$?
10. Is B = $(A + B_{\text{second pair}}) \div 2$?

**Type Examples with Solutions:**

**Example 1:**
$24 : 48 :: 36 : ?$
Solution: $24 \times 2 = 48$, so $36 \times 2 = 72$

**Example 2:**
$81 : 9 :: 64 : ?$
Solution: $\sqrt{81} = 9$, so $\sqrt{64} = 8$

**Example 3:**
$2 : 8 :: 5 : ?$
Solution: $2^3 = 8$, so $5^3 = 125$

**Example 4:**
$3 : 30 :: 4 : ?$
Solution: $3 \times 10 = 30$, so $4 \times 10 = 40$

**Example 5 (Complex):**
$6 : 35 :: 9 : ?$
Solution: $6^2 - 1 = 35$, so $9^2 - 1 = 80$

**тЪб NCEE Common Mistakes:**

1. Assuming multiplication when it's actually division
2. Mixing up square and cube relationships
3. Not checking if the operation works for the second pair before answering
4. Rushing тАФ always verify your answer
