# Algebraic Expressions & Simple Equations

### рҹҹў Lite вҖ” Quick Review (1hвҖ“1d)
> Rapid summary for last-minute revision before your exam.

Algebraic expressions combine numbers and variables using addition, subtraction, multiplication, and division. An equation states that two expressions are equal and can be solved to find the unknown variable's value.

For linear equations in one variable, the goal is always to isolate the variable on one side. For 4x вҲ’ 9 = 23: add 9 to both sides вҶ’ 4x = 32; divide by 4 вҶ’ x = 8. Always perform the same operation on both sides to maintain equality.

Key algebraic identities to memorise:
- (a + b)ВІ = aВІ + 2ab + bВІ
- (a вҲ’ b)ВІ = aВІ вҲ’ 2ab + bВІ
- aВІ вҲ’ bВІ = (a + b)(a вҲ’ b)
- (a + b)Ві = aВі + 3aВІb + 3abВІ + bВі

вҡЎ **Exam tip:** When solving word problems, define the unknown clearly first. If the problem says "three consecutive integers sum to 48," let the numbers be n, n+1, n+2, giving 3n+3 = 48 вҶ’ n = 15. The numbers are 15, 16, 17.

вҡЎ **Exam tip:** Always check your solution by substituting back into the original equation. This catches algebraic errors before they cost marks.

**Linear Equations in One Variable**

A linear equation has the form ax + b = c, where a вү  0. The solution process involves inverse operations applied in reverse order of PEMDAS (reverse PEMDAS: subtract/add, then divide/multiply).

Solve 7(x вҲ’ 3) = 2x + 12:
1. Expand: 7x вҲ’ 21 = 2x + 12
2. Collect x terms: 7x вҲ’ 2x = 12 + 21
3. Simplify: 5x = 33
4. Divide: x = 33/5 = 6.6

**Systems of Linear Equations**

Two equations with two unknowns can be solved by substitution or elimination.

Substitution method: From equation 1: y = 3x вҲ’ 7. Substitute into equation 2: 2x + 3(3x вҲ’ 7) = 8 вҶ’ 2x + 9x вҲ’ 21 = 8 вҶ’ 11x = 29 вҶ’ x = 29/11. Then y = 3(29/11) вҲ’ 7 = 87/11 вҲ’ 77/11 = 10/11.

Elimination method: Multiply equation 1 by 2 and equation 2 by 3, then subtract to eliminate x. The substitution method is often faster when one variable already has a simple coefficient.

**Algebraic Expressions вҖ” Manipulation Skills**

Factorisation is the reverse of expansion. Key techniques:

Factorising quadratic expressions where a = 1: xВІ + 5x + 6 = (x + 2)(x + 3) because 2 + 3 = 5 (coefficient of x) and 2 Г— 3 = 6 (constant).

Factorising by grouping: ax + ay + bx + by = a(x + y) + b(x + y) = (a + b)(x + y). For 2x + 2y + 3x + 3y: group as 2(x + y) + 3(x + y) = 5(x + y).

**Common Mistakes to Avoid:**

| Mistake | Correct approach |
|---------|-----------------|
| Forgetting to multiply every term when expanding brackets | (x + 3)(x вҲ’ 4) = xВІ вҲ’ 4x + 3x вҲ’ 12 = xВІ вҲ’ x вҲ’ 12 |
| Cancelling terms instead of factors | Can only cancel factors: (x+2)/x+2 is NOT 1, but (x+2)/x+2 = 1 is wrong; must cancel the WHOLE numerator against denominator |
| Changing signs incorrectly when moving terms | Whatever you do to one side, you MUST do to the other |
| Not simplifying the final answer | xВІ вҲ’ 4 can be factorised to (x+2)(x-2); always factorise if possible |

**The Fundamental Theorem of Algebra and Simple Equations**

While the fundamental theorem states that a polynomial of degree n has exactly n complex roots, our focus is on practical solution methods for equations encountered in entrance exams.

**Word Problem Strategies**

Translating word problems into algebraic equations is a critical skill. Consider: "The sum of three times a number and 17 is 74. What is the number?" Translation: 3x + 17 = 74 вҶ’ 3x = 57 вҶ’ x = 19.

More complex example: "A father is four times as old as his son. In 20 years, he will be twice as old. Find their current ages." Let son's age = x, father's age = 4x. In 20 years: 4x + 20 = 2(x + 20) вҶ’ 4x + 20 = 2x + 40 вҶ’ 2x = 20 вҶ’ x = 20. Son is 20, father is 80.

Check: In 20 years, son is 40, father is 100. Father is 2.5 times, not twice. Wait вҖ” let me recalculate: 4x + 20 = 2(x + 20) вҶ’ 4x + 20 = 2x + 40 вҶ’ 2x = 20 вҶ’ x = 10. Son is 10, father is 40. In 20 years: son 30, father 60. Yes, father is twice son's age. Correct.

**Partial Fractions**

For rational expressions, partial fraction decomposition can simplify integration and equation solving. For 5x + 3 / (x вҲ’ 1)(x + 2), write as A/(x вҲ’ 1) + B/(x + 2). Multiply both sides by (x вҲ’ 1)(x + 2): 5x + 3 = A(x + 2) + B(x вҲ’ 1). Solve: let x = 1 вҶ’ 8 = 3A вҶ’ A = 8/3. Let x = вҲ’2 вҶ’ вҲ’7 = вҲ’3B вҶ’ B = 7/3.

**Absolute Value Equations**

Equations with absolute values require considering both cases. |2x вҲ’ 3| = 7 gives two cases: 2x вҲ’ 3 = 7 вҶ’ 2x = 10 вҶ’ x = 5, and 2x вҲ’ 3 = вҲ’7 вҶ’ 2x = вҲ’4 вҶ’ x = вҲ’2.

For |x + 1| < 5, this means вҲ’5 < x + 1 < 5, so вҲ’6 < x < 4.
